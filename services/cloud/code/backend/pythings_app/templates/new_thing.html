New Thing


You can add new Things by installing PythingsOS on them or by running Simulator.

To install PythingsOS on the Esp8266 and on the Esp32, you can use the self-contained installer, which allows to get you started straight away without worrying about tools, dependencies and firmwares.

You can also create a Thing simulator to immediately give Pythings a try, without any hardware. A Docker image is ready to use! To install on other platforms, you will need to perform a manual install, you can read more about it in the documentation.
